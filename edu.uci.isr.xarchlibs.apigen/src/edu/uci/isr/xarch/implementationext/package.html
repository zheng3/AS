DOM-Based implementation of the implementationext xArch XML Schema.

This package contains generic interfaces for the implementationext xArch XML schema and DOM-based implementations of those interfaces. To use this package, users should first create an IXArch object using the methods in edu.uci.isr.xarch.XArchUtils. Then, they should create a implementationextContext object using that IXArch. The context object can be used to create objects in this namespace.
